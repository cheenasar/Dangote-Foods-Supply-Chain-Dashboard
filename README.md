# Dangote Foods Nigeria — Supply Chain Performance Dashboard

 Project Overview
This project analyzes the Q1 2025 supply chain performance of Dangote Foods Nigeria 
across three months — January, February and March. The dashboard tracks key supply 
chain KPIs to identify strengths, weaknesses and areas for improvement.

Problem Statement
Management needed a clear view of supply chain performance across Q1 2025 to 
identify delivery inefficiencies and rising return rates affecting customer satisfaction.

Tools Used
- Microsoft Power BI — Dashboard building and visualization
- Microsoft Excel — Data storage and preparation
- SQL — Data querying and KPI calculation
- DAX — Calculated measures in Power BI

KPIs Analyzed
- Order Fulfillment Rate
- On Time Delivery Rate (OTD)
- Return Rate
- Inventory Turnover Ratio
- Cost Per Order

 Key Findings
- Order Fulfillment Rate consistently met the 95% benchmark across all 3 months
- OTD Rate was consistently at 92% — below the 95% industry benchmark
- Return Rate increased from 2.5% in January to 3% in March - a worrying trend
- Inventory Turnover remained stable at 5x — indicating efficient inventory management
- Cost Per Order remained stable at ₦2,000 despite 50% growth in order volumes

Recommendations
- Investigate root cause of consistently low OTD rate across all 3 months
- Review logistics partner contracts and enforce delivery SLA clauses
- Monitor rising return rate — likely connected to late deliveries
- Consider onboarding additional logistics partners to improve delivery capacity
